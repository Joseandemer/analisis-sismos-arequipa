# Análisis de Sismos en Arequipa (Volcán Misti)

Herramientas utilizadas:
- Excel
- SQL / MySQL
- Power BI

Descripción:
Proyecto de análisis de datos enfocado en el estudio de
eventos sísmicos en la zona de Arequipa, evaluando frecuencia,
magnitud y patrones asociados al volcán Misti mediante
análisis de datos y visualización.
